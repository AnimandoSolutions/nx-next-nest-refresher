# Local Development

## Local Environment set up

### Prerequisites

- Docker

### Run docker-compose

```bash
docker compose up -d
```

## Run the app

```bash
pnpm nx run-many -t serve
```

## Run unit tests

```bash
pnpm nx run-many -t test
```

## Run e2e tests

```bash
pnpm nx run-many -t e2e
```
